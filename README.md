# Authentication Service using Spring Boot & JWT

##  Description
This project implements a basic authentication system using Spring Boot. It provides secure login and signup functionality using JWT (JSON Web Tokens).

##  Features
- User Registration and Login
- JWT-based Authentication
- Secure API endpoints
- Token-based session handling

##  Tech Stack
- Java
- Spring Boot
- Spring Security
- JWT

## 🔗 Backend Concepts
- REST API development
- Authentication and Authorization
- Token-based security (JWT)
